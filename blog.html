Important FAQ?

What is the Document Object Model (DOM) ?

DOM is the data representation of objects that comprise the structure and content of a document on the web.

How do you select an element from the DOM ?

To select all an element with the same tag name with JavaScript, I can use the document. getElementsByTagName() method, which takes the tag name as a parameter and returns a collection of elements.

What is event delegation in the context of the DOM, and why is it useful ?

Event delegation is a technique used in JS and DOM manipulation where instead of attaching event listeners to individual elements, we attach a single event listener to a parent element and rely on the event bubbling mechanism to catch events on child elements. Event delegation is very useful because of its improved performance, Dynamic content handling and simplified event handling.

How do you manipulate on element`s attributes and styles using the DOM ?

Manipulating DOM elements is a fundamental aspect of web development, allowing developers to dynamically alter the content, structure, and appearance of a webpage using JavaScript. The classList object only provide methods to change the class of an element. If I want to change other attributes like id , href , or src , I can use the setAttribute() method.
